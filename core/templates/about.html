What is JobSet?

JobSet is a job and internship portal designed especially for students and freshers. It helps users register, create a profile, and explore job listings with ease.

Why This Project Was Built

This project was developed as part of my BCA curriculum to showcase my understanding of web development using HTML, CSS, and Python (Django). The aim was to create a practical and user-friendly job portal specifically for students and freshers.
Through JobSet, I wanted to demonstrate how real-world applications like user login, profile handling, and job searching can be built with simple technologies.
